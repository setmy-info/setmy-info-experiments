## Rules

* Every neuron has its own bias.
* Neuron input are weights and bias.

Where:

o = s((Σi[n]w[n]) + b)

$$ o = σ\left(\sum_{n} i_n w_n + b\right) $$
- 

- i = input
- o = output
- w = weight
- b = bias
- n = index, element number
- s, σ = sigmoid function
